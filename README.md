# PRODIGY_GA_04

## Image-to-Image Translation

This project demonstrates a simple image-to-image translation using Python and Google Colab as part of the Prodigy InfoTech Generative AI Internship.

### Technologies Used
- Python
- Google Colab
- Pillow (PIL)
- Matplotlib

### Description
This project takes an input image and applies an image transformation technique to generate a translated output image. It demonstrates the basic concept of image-to-image translation using Python.

### Input
An input image (e.g., Apple, Robot, Flower, etc.)

### Output
Displays the original image and the translated image side by side after applying the transformation.
